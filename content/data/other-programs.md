## Other Treasury Programs Data

This page provides access to data from various Treasury programs and initiatives not covered in other data sections.

### Available Data Sets

#### Financial Assistance Programs
Data on Treasury-administered financial assistance programs including:
- Grant programs
- Loan programs
- Investment programs
- Technical assistance

#### Regulatory Programs
Information from Treasury regulatory activities:
- Rulemaking data
- Compliance statistics
- Enforcement data

#### Historical Data
Archives of data from completed programs:
- TARP data
- Recovery Act programs
- Special temporary programs

### Accessing the Data

Treasury program data is available through:
- [USAspending.gov](https://www.usaspending.gov/) - Federal spending data
- [Fiscal Data](https://fiscaldata.treasury.gov/) - Treasury financial data
- [Data.gov](https://www.data.gov/) - Open government data

### Data Formats

Data is typically available in:
- CSV (Comma Separated Values)
- Excel spreadsheets
- API access for some datasets
- PDF reports
